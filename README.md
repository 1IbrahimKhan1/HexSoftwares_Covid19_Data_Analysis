# COVID-19 Data Analysis Using Python

## Project Overview

This project analyzes global COVID-19 data using Python and popular data science libraries such as Pandas, NumPy, and Matplotlib. The objective is to explore trends in confirmed cases, deaths, recoveries, active cases, and regional distributions through data visualization and exploratory data analysis (EDA).

## Dataset

The dataset used in this project contains country-wise COVID-19 statistics, including:

* Confirmed Cases
* Deaths
* Recoveries
* Active Cases
* New Cases
* New Deaths
* WHO Region Information

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Dataset Exploration
4. Statistical Analysis
5. Visualization of COVID-19 Trends
6. WHO Region Analysis
7. Correlation Analysis
8. Insights and Conclusion

## Visualizations Included

* Top 10 Countries by Confirmed Cases
* Top 10 Countries by Deaths
* Top 10 Countries by Recoveries
* Top 10 Countries by Active Cases
* WHO Region-wise Confirmed Cases
* New Cases vs New Deaths Scatter Plot
* Correlation Heatmap

## Key Insights

* Countries with higher confirmed cases generally reported higher death counts.
* Recovery rates varied significantly across countries.
* Certain WHO regions were more heavily affected than others.
* Strong correlations were observed between confirmed cases, active cases, and deaths.

## Conclusion

This project demonstrates fundamental data science skills including data cleaning, exploratory data analysis, and data visualization. It provides valuable insights into the global impact of the COVID-19 pandemic and showcases practical applications of Python in real-world data analysis.
